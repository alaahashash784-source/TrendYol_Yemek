# TrendYol Yemek - Online Sipariş Sistemi

## Proje Hakkında
Bu proje, ASP.NET MVC kullanarak yapılmış bir yemek sipariş uygulamasıdır.

## Özellikler
- Kullanıcı kayıt ve giriş sistemi
- Restoran listesi görüntüleme
- Yemek sipariş verme
- Sipariş geçmişi
- Admin paneli

## Teknolojiler
- ASP.NET MVC 5
- Entity Framework 6
- SQL Server
- Bootstrap 5
- jQuery

## Nasıl Çalıştırılır
1. Visual Studio'da açın
2. NuGet paketlerini yükleyin
3. Veritabanını oluşturun (create_db.sql)
4. Projeyi çalıştırın

## Veritabanı
SQL Server kullanılmaktadır. Bağlantı ayarları Web.config dosyasındadır.

## Dosya Yapısı
- Controllers/ - MVC Controller'lar
- Models/ - Veritabanı modelleri
- Views/ - Görünüm dosyaları
- Content/ - CSS dosyaları
- Scripts/ - JavaScript dosyaları

## Admin Girişi
Admin paneline erişim için /Admin/Login adresini kullanın.
